# Contributing

Thanks for contributing to `log-tidy`.

## Local setup

```bash
npm install
npm run build
npm test
```

## Pull requests

- Keep changes scoped and covered by tests.
- Update `docs/log-format.md` when emitted fields change.
- Update `docs/operations.md` when operational behavior changes.
- Call out security-sensitive behavior in the PR description.

## Issues

For bug reports, include a small input sample, expected output, actual output,
and the affected version. Do not include real secrets.
